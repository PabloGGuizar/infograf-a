🚀 Guía para ejecutar tu aplicación

1. Crear clave API

Accede a Google AI Studio y genera tu clave API. No la compartas con nadie.

2. Abre tu proyecto

Arrastra la carpeta del proyecto sobre el ícono de Visual Studio Code.

3. Configura la clave API

Pega tu clave en el archivo .env.local.

4. Instala dependencias

Abre la terminal y escribe: npm install.

5. Ejecuta la aplicación

Escribe en la terminal: npm run dev.

6. Abre tu aplicación

Ve al enlace local que aparece en la consola.

7. Detén la ejecución

Presiona Ctrl + C en la terminal para detener la aplicación cuando termines.
